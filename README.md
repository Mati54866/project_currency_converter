# Currency Converter 💱

A simple, real-time currency converter application built with React. Convert between different currencies with live exchange rates.

## ✨ Overview

This Currency Converter app allows users to convert amounts between multiple currencies using up-to-date exchange rates. Features a clean, intuitive interface for quick conversions.

## 🛠️ Tech Stack

- **React** - Frontend library
- **CSS3** - Styling
- **Exchange Rate API** - Live currency data

## 🔋 Key Features

- ✅ Real-time currency conversion
- ✅ Support for multiple currencies
- ✅ Swap currencies with one click
- ✅ Responsive design
- ✅ Clean, user-friendly interface

## 🚀 Quick Start

### Prerequisites
- Node.js installed
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Mati54866/project_currency_converter.git

# Navigate to project
cd project_currency_converter

# Install dependencies
npm install

# Start development server
npm start
